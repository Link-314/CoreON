# CoreON

CoreON is a lightweight browser-based 3D engine and scene editor inspired by Blender. It allows you to create, edit, and export interactive 3D scenes directly in the browser using WebGL and Three.js.

---

## Features

* 3D scene editor
* Object transform controls
* Materials and colors
* Lighting system
* JSON scene export/import
* Multiple primitive objects
* Camera controls
* Real-time rendering
* Lightweight interface
* Fully browser-based

---

## Supported Objects

* Cube
* Sphere
* Plane
* Cylinder
* Cone
* Torus
* Lights

---

## Scene Format

CoreON uses a custom JSON scene format.

Example:

```json
{
  "version": "1.0",
  "engine": "CoreON",
  "objects": []
}
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Link_314/CoreON.git
```

Open the project folder:

```bash
cd CoreON
```

Run a local server or open `index.html` directly in the browser.
